Liquid Web Design

Liquid design gives the user control over the webpage and text sizing. The liquid design means that you never have to scroll from left to right. All elements on the page will resize to fit onto the computer screen no matter the screen size.

Fixed Web Design

the web browser will not be able to change the format text of the design as they see fit. This is the most popular and common option for website design. With this design the width of the content is contained and stays fixed no matter what.

F-Shaped Web Design

The F-Pattern describes the most common user eye-scanning patterns when it comes to blocks of content. 'F' means fast. That's how users read your content on the web. In a few seconds, users eyes move at amazing speeds across a' page.

Z-Shaped Web Design

It suggests that most readers eyes will scan your page starting in the top-left, move horizontally to the top-right then diagonally to the bottom-left before finishing with another horizontal movement to the bottom-right, hence forming an imaginary “Z” shape.
